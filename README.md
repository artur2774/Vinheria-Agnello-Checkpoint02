Sistema de cadastro e controle de estoque de vinhos

Foram implementados novos de JavaScript, CSS e HTML
Como por exemplo: funções, estruturas de repetição, sistemas de validação de entrada de dados. Além de conceitos de efeitos e animação com o CSS

Esse trabalho tem o objetivo de criar um estoque com diversas indetifcações como nome, safra e tipo. Exibindo o status do estoque como: quantidade total, nível (baixo, médio e alto), separação em faixas de safra (jovem, amadurecido e antigo)
